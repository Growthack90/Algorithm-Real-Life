# DISTRIBUTORE AUTOMATICO

## Problema
Creare logica di flusso che emula un distributore automatico di prodotti.

## Soluzione
Work-flow:
- scegli prodotto
- Mostra prezzo
- Inserisci moneta
- Verifica valore moneta
- Erogare prodotto se moneta >= Prezzo prodotto
- Dare eventuale resto del credito residuo
